 Parcial 1

Este repositorio contiene el código del Parcial 1 de Programación con Tecnologías Web
- React: Librería principal para la construcción de la interfaz de usuario.
- react-i18next: Librería utilizada para la internacionalización de la aplicación.
- Boostrap: para los estilos de los componentes

Instalación

Para ejecutar el proyecto localmente, sigue estos pasos:

1. Clonar el repositorio:

   git clone https://github.com/IavilaGw/parcial1.git

npm install
npm start
